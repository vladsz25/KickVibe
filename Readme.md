# AI assisted software development

Repo for final projects developed within the course "AI assisted software development" held at UVT.

## Projects ideas

*Each project requires implementing features like user authentication, database operations (localStorage or API integration), CRUD functionality, responsive design, form handling, state management, search/filter capabilities, and data visualization.*
